# Cellular Containment

Conway's Game of Life staged as a biocontainment lab's live culture-observation console — a circular petri-dish grid, hazard-tape signage, colony age/size/shape tracking, day/UV lab lighting, and switchable mutation protocols (alternate birth/survival rulesets).

**Live page:** https://eremin-andrey.github.io/game-of-life-biohazard/

## Features

- Standard Conway rules (B3/S23) plus five alternate "mutation protocols": HighLife, Day & Night, Seeds, Replicator, and Diamoeba
- Colony age tracked via cell color and size — fresh growth vs. dormant colony
- 20 irregular colony silhouettes, randomly assigned on birth
- Day / UV (blacklight) observation lighting modes
- Click or drag on the dish to inoculate cells by hand
- Specimen library of classic patterns: Glider, Pulsar, R-Pentomino, Acorn, Gosper Glider Gun

## Usage

Open `index.html` (or the live page above) in any modern browser — no build step, no dependencies.
